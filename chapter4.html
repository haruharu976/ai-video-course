基本的な動画編集

まずは動画のカット・結合・書き出しを体験する。

PYTHON — BASIC 
from moviepy.editor import VideoFileClip, concatenate_videoclips

# 動画を読み込み
clip = VideoFileClip("input.mp4")

# 最初の10秒だけ切り出す
clip_cut = clip.subclip(0, 10)

# 別の動画と結合
clip2 = VideoFileClip("input2.mp4")
final = concatenate_videoclips([clip_cut, clip2])

# MP4として書き出し
final.write_videofile("output.mp4")

テキスト・ロゴ合成

動画にテキストやロゴを重ねる。SNSのリール動画テンプレートの基本。

PYTHON — TEXT OVERLAY 
from moviepy.editor import VideoFileClip, TextClip, CompositeVideoClip

clip = VideoFileClip("input.mp4")

# テキストクリップ作成
txt = TextClip(
  "AIで動画制作",
  fontsize=70, color='white',
  font='Noto-Sans-CJK-JP'
).set_position('center').set_duration(3)

# 動画に合成
final = CompositeVideoClip([clip, txt])
final.write_videofile("output.mp4")

STEP 04

CSVから動画を大量生成

CSVのデータをループで流し込んでパーソナライズ動画を自動量産する。SNSマーケの核心。

PYTHON — BULK GENERATE 
import csv
from moviepy.editor import VideoFileClip, TextClip, CompositeVideoClip

# CSVからデータ読み込み
with open('data.csv') as f:
  rows = list(csv.DictReader(f))

base = VideoFileClip("template.mp4")

for row in rows:
  txt = TextClip(row['name'], fontsize=60, color='white')
  txt = txt.set_position('center').set_duration(base.duration)
  final = CompositeVideoClip([base, txt])
  final.write_videofile(f"out_{row['id']}.mp4")
  print(f"✅ {row['name']} 完了")
📋 CSVの例（data.csv）

id,name
1,田中様
2,鈴木様
3,佐藤様

  • CSVファイルを用意した
  • ループで複数の動画が生成された
  • Codexで独自のデータ構造に合わせたスクリプトを生成した

STEP 05

自動字幕生成

Whisperでテキスト化した文字起こしをMoviePyに流し込み、単語ごとに色が変わる動的字幕を自動生成する。

PYTHON — AUTO CAPTIONS 
import whisper
from moviepy.editor import VideoFileClip, TextClip, CompositeVideoClip

# Whisperで文字起こし
model = whisper.load_model("base")
result = model.transcribe("video.mp4", word_timestamps=True)

clip = VideoFileClip("video.mp4")
subtitles = []

for seg in result['segments']:
  txt = TextClip(
    seg['text'], fontsize=40,
    color='yellow', stroke_color='black', stroke_width=2
  ).set_start(seg['start']).set_end(seg['end'])
  txt = txt.set_position(('center', 'bottom'))
  subtitles.append(txt)

final = CompositeVideoClip([clip] + subtitles)
final.write_videofile("captioned.mp4")
pip3 install openai-whisper
  • whisperをインストールした
  • 自分の動画に字幕を自動生成した

// CAUTION

受講前に知っておくべき注意点

🔑
APIキーは絶対にコードに直書きしない
OpenAI APIキーやAWS認証情報を .js や .py ファイルに直接書くと、GitHubにpushした瞬間に世界中に公開される。必ず .env ファイルに書いて、.gitignore に追加する。
🗑️
AIが生成したコードをすぐに本番環境で実行しない
特にExtendScriptとMoviePyはファイルの上書き・削除を行う可能性がある。必ずテスト用のコピーフォルダで試してから本番素材に適用する。
💸
AWS Lambdaのコスト管理（CH.02 Remotion）
Remotionのクラウドレンダリング（Lambda）はAWSの従量課金。上限アラートを必ず設定してから使い始める。ローカルレンダリングのみなら追加費用ゼロ。最初はローカルで動作確認を完了させること。
🤖
AIのハルシネーション（存在しない関数を書くことがある）
CodexやClaudeが存在しないMoviePyのメソッドやAEのエクスプレッション関数を生成することがある。エラーが出たらそのエラーメッセージをそのままAIに「このエラーを修正して」と渡すデバッグループを習慣にする。
📁
AGENTS.md の使い分け
このコースではAGENTS.mdをプロジェクトのルートに保存する。Codexが自動で読み込み、毎回ルールを守った状態でコードを生成してくれる。
※ Claude Codeを使う場合は同じ内容を CLAUDE.md として保存する。
🗺️

// SECURITY

.env ファイルの作り方（必須）

CH.05でClaude APIを使う前に、必ずこの手順でAPIキーを管理する。

.env プロジェクトのルートフォルダに保存
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxxxxxxx # AWS Lambda（Remotion クラウドレンダリング用・任意） AWS_ACCESS_KEY_ID=your_access_key AWS_SECRET_ACCESS_KEY=your_secret_key AWS_REGION=ap-northeast-1
.gitignore 同じフォルダに保存（GitHubへの流出防止）
# 絶対にGitにコミットしないファイル .env .env.local .env.production # 出力ファイル（大容量） out/ *.mp4 *.mov __pycache__/ node_modules/
💡
Pythonでの読み込み方
pip3 install python-dotenv でライブラリをインストール後、スクリプト冒頭に以下を追加：
from dotenv import load_dotenv; load_dotenv()
これで os.environ["ANTHROPIC_API_KEY"] で安全に読み込める。

受講前チェックリスト

講義開始前にすべてにチェックが入っている状態にしておこう。

  • ターミナル（Mac）またはコマンドプロンプト（Windows）が開ける
  • node -v でv18以上が表示される（CH.02用）
  • python3 --version で3.8以上が表示される（CH.04・05用）
  • pip3 install moviepy が完了している（CH.04用）
  • After Effects が起動できる（CH.03用）
  • OpenAI APIキー を取得して .env に保存済み（CH.05用）
  • VS Code または任意のエディタ がインストールされている
  • テスト用のMP4ファイル を1本用意している（CH.04用）
  • APIキーは .env に保存し、.gitignore に .env を追加している
  • AGENTS.md をプロジェクトフォルダに作成した

エラーが出たら AI に相談する

セットアップ中にエラーが出ても焦らなくてOK。エラーメッセージをそのままコピーして、以下のテンプレートでAIに質問するだけでほぼ解決できる。

📋 AI相談テンプレート（コピーして使う）
// TEMPLATE
Remotionのセットアップ中に以下のエラーが出ました。解決方法を教えてください。 【環境】 - OS: Mac（またはWindows） - Node.js: （node -v の結果を貼る） - 実行したコマンド: （例：npx create-video@latest my-video） 【エラーメッセージ】 （ターミナルのエラー全文をここに貼り付ける）

テンプレートをコピーして、エラーメッセージを貼り付けてから以下のAIに送信：

→ Claude で相談 → ChatGPT で相談
💡
よくあるエラーと対処法
npm error 404 create-remotion → 正しいコマンドは npx create-video@latest
permission denied → Macの場合 sudo を先頭につけて再実行
command not found: node → nodejs.org からLTS版をインストール
ENOSPC / disk space → ディスク空き容量を確認（最低5GB以上推奨）

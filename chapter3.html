② 新規コンポジションを作成する

コンポジション＝動画1本分の「キャンバス」のこと。サイズとフレームレートをここで決める。

または Ctrl+N（Mac: ⌘+N）
2

プリセット HDTV 1080 29.97 を選択

💡 フレームレート（fps）とは？

1秒間に何枚の画像を表示するかの数字。30fps = 1秒間に30枚。映像がなめらかに見える基準値。SNS動画は30fpsが一般的。

💡 1920×1080とは？

画面の横×縦のピクセル数。フルHDと呼ばれる標準的なサイズ。YouTubeやInstagramリールに最適。

④ エクスプレッション入力欄を開く

これが次のステップの準備。タイムラインのプロパティに「式（エクスプレッション）」を入力できる場所を開く。

1
タイムラインのレイヤー左の「▶」をクリック
レイヤー名の左にある三角アイコンを押すとプロパティが展開される
2
「変形」→「不透明度」を探す
テキスト → 変形 → 不透明度 と階層が続く
3
「不透明度」の文字を Alt+クリック（Mac: Option+クリック）
タイムラインにオレンジ色の入力欄が現れる → これがエクスプレッション入力欄
⚡ ここにコードを入力すると動きが制御できる
⚠️
クリックしただけでは開かない。必ずAlt（Option）+クリック。普通にクリックするとキーフレームが打たれるだけなので注意。

STEP 02

Expressionの基礎

エクスプレッションはレイヤープロパティに直接書くJavaScript。Alt+クリックでプロパティに入力欄が現れる。

EXPRESSION BASIC 
// 現在の時間（秒）を取得
time

// フレーム数を取得
timeToFrames(time)

// 0〜100の間をループ
linear(time % 2, 0, 2, 0, 100)

// wiggle(周波数, 振れ幅)
wiggle(3, 50)

最初に試すエクスプレッション：wiggle
不透明度プロパティのエクスプレッション欄に入力すると、ランダムに揺れ動くアニメーションができる。

wiggle(3, 30)
  • 不透明度にwiggleを入力して動きを確認した
  • 数字を変えて動きの違いを体感した

Codexで Expression生成

自然言語の指示をCodexに入力すると、複雑なエクスプレッションを自動生成してくれる。

💬 あなたの指示
「ヌルオブジェクトとの距離に応じて不透明度を0〜100で変化。500px以上で完全透明に」
⚡ 生成されたコード
var n=thisComp.layer("Null 1");
var d=length(
  n.toComp([0,0]),
  toComp([0,0])
);
linear(d,0,500,100,0)

下のフォームに情報を入力すると、あなたのAEプロジェクト専用のエクスプレッションプロンプトが生成される。そのままCodexに貼り付けてコードをもらおう。

🎬 AE EXPRESSION GENERATOR 入力 → 生成 → AEに貼り付け

どんな動きにしたいか *
数値の範囲（任意）
レイヤー名・参照オブジェクト（任意）
💬 生成されたプロンプト（Codexに貼り付け）
💡 次のステップ

生成されたプロンプトをCodexに送る → 返ってきたコードをAEのエクスプレッション欄（Alt+クリック）に貼り付ける → エラーが出たらエラーメッセージごとAIに渡して「このエラーを修正して」と伝える。

  • フォームに自分のやりたいことを入力してプロンプトを生成した

STEP 04

ExtendScript 自動化

ExtendScriptはAE全体を操作できる自動化スクリプト。ファイル → スクリプト → スクリプトエディターから実行できる。

EXTENDSCRIPT 
// 全レイヤーにエフェクト適用 + ファイル名テキスト追加
var comp = app.project.activeItem;
for(var i=1;i<=comp.layers.length;i++){
  var l=comp.layers[i];
  if(l instanceof AVLayer){
    l.Effects.addProperty("Brightness & Contrast");
    var t=comp.layers.addText(l.source.name);
    t.transform.position.setValue([comp.width/2,comp.height/2]);
  }
}
コードをコピーしてスクリプトエディターに貼り付け
💡 スクリプトエディターの開き方

AE上部メニュー → ファイル → スクリプト → スクリプトエディター
コードを貼り付けて「スクリプトを実行」ボタンを押すだけ。

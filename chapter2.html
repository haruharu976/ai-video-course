STEP 02

環境構築

Remotionに必要なのはNode.jsのみ。すでにインストール済みの場合はバージョン確認だけでOK。

// Node.jsが必要な理由

① Node.jsのバージョンを確認

node -v
✅ こうなったらOK
$ node -v
v20.11.0
# v18.x.x や v22.x.x でもOK。18以上ならすべて対応。
❌ エラーが出た場合
パターン① — Node.jsが入っていない
command not found: node
# → 下の「② Node.jsが入っていない場合」でインストール

パターン② — バージョンが古い
$ node -v
v16.14.0
# → v18未満はNG。nodebrewかnvmでアップデートが必要。

② Node.jsが入っていない場合

brew install node # macOS

③ npxが使えることを確認

npx -v
✅ こうなったらOK
$ npx -v
10.2.4
# バージョン数字が表示されればOK（数字は違っていい）
  • ターミナルを開いた
  • node -v でv18以上が表示された
  • npx -v でバージョンが表示された

STEP 03

プロジェクト作成

コマンド1本でRemotionプロジェクトが生成される。

// コマンド実行後に作られるフォルダ構成

① ターミナルを開く

// Mac / Windows それぞれの開き方

② プロジェクトを作る場所に移動する

cd ~/Desktop # 任意の場所（my-videoフォルダをどこに作るかの話。デスクトップが一番わかりやすいのでおすすめ。Documents や自分で作ったフォルダでもOK。場所はどこでも動作に影響しない。）
✅ こうなったらOK（何も表示されなければ成功）
$ cd ~/Desktop
$  # ← 何も出ずに次の $ が表示されればOK。エラーなし＝成功。

② Remotionプロジェクトを初期化

npx create-video@latest my-video
📋 途中で聞かれること

テンプレート選択 → Blank を選択してEnter。
TailwindCSS / agent skills は任意（どちらでもOK）。

③ フォルダに入って依存パッケージをインストール

cd my-video && npm install
  • npx create-video@latest が完了した
  • my-videoフォルダが作成された
  • npm install が完了した（node_modulesが作られた）

STEP 04

Remotion Studio 起動

Remotion Studioはブラウザ上で動くリアルタイムプレビュー環境。コードを変更すると即座に映像が更新される。

// Remotion Studio の画面（こんな感じで開く）

① Studioを起動

npx remotion studio
🌐 自動でブラウザが開く

開かない場合は http://localhost:3000 にアクセス。
左側にコンポジション一覧、中央にプレビューが表示される。

② コードを書き換えて即時反映を体験

src/Composition.tsx を編集してみる 
import { useCurrentFrame, interpolate } from 'remotion'

export const MyComposition = () => {
  const frame = useCurrentFrame()
  const opacity = interpolate(frame, [0, 30], [0, 1])

  return (
    <div style={{ opacity, color: 'white', fontSize: 80 }}>
      Hello AI 👋
    </div>
  )
}

30を10に変えて保存 → プレビューが即更新されることを確認！

  • Remotion Studioがブラウザで開いた
  • プレビュー画面が表示された
  • コードを変更してリアルタイム更新を体験した

STEP 05

Codexと接続 → MP4出力

AIにプロンプトを渡してコードを生成し、コマンド1本でMP4として書き出す。これがゴール。

// この章のゴール：プロンプト → MP4 の全体像

① Remotion Skillsをインストール（推奨）

npx -y skills@latest add remotion-dev/skills -g -y
💡 Remotion Skills とは

AIエージェントにRemotionの使い方を教えるコンテキストファイル。これを入れるとCodexの精度が大幅に向上する。

② Codexにこのプロンプトを送る

💬 CODEX PROMPT EXAMPLE（コピーして使う）
「中央に白いテキスト『Hello AI』を表示して。0フレームから徐々にフェードインし、1秒でズームインしながら現れる。背景はダークブルーのグラデーション。30fps、5秒の動画。」

③ 生成されたコードをComposition.tsxに貼り付けてStudioで確認

④ 動画をMP4として出力

npx remotion render MyComp out/video.mp4
🎬 出力完了

プロジェクトの out/video.mp4 に動画が生成される。
これであなたのAI生成動画第1号が完成！

  • Remotion Skillsをインストールした
  • CodexにプロンプトでRemotionコードを生成した
  • Studioでプレビューを確認した
  • npx remotion render でMP4が出力された
  • 🎉 最初のAI生成動画が完成！

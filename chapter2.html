STEP 01

料金・ライセンス確認

まず自分が無料で使えるかどうかを確認する。Remotionは個人・小規模利用は完全無料。

プラン料金対象
Free License 無料 個人・学習・スタートアップ（従業員3名以下）
Company License $25/人/月〜
最低$100/月 法人利用・商用プロダクト
Enterprise 要相談 大規模利用・カスタム契約
💡 AWS Lambda（クラウドレンダリング）

ライセンスとは別に、クラウドレンダリングはAWS従量課金。軽量な動画1本あたり約$0.001〜$0.02程度。ローカルレンダリングのみなら追加費用ゼロ。

⚠️ 料金は変動する場合があります。最新情報は remotion.dev/docs/license で確認してください。

⚠️ 従量課金について

このチャプターでは Codex（OpenAI）を使ってRemotionコードを生成します。
Codexの利用料金は以下の通りです：

ChatGPT Plus ($20/月) Codex使用量が月額料金に含まれる（上限あり）
APIキー直接利用 gpt-5.3-codex: 入力$1.75/百万token・出力$14.00/百万token
1セッションの目安: $0.50〜$2.00程度
Remotion Lambda ローカルレンダリングのみなら追加費用ゼロ
クラウドレンダリング使用時: 動画1本あたり約$0.001〜$0.02

💡 このコースでは基本的にローカルレンダリングを使います。Codexも最初はChatGPT Plusの範囲内で十分です。

STEP 02

環境構築

Remotionに必要なのはNode.jsのみ。すでにインストール済みの場合はバージョン確認だけでOK。

① Node.jsのバージョンを確認

node -v
✅ 必要バージョン

Node.js v18以上が必要。macOSはSequoia 15以上が必須。
v18.x.x や v20.x.x と表示されればOK。

② Node.jsが入っていない場合

brew install node # macOS (Homebrew)
💡 Homebrewがない場合

nodejs.org からインストーラをダウンロードしてもOK。LTS版を選択してください。

③ npxが使えることを確認

npx -v
  • ターミナル（またはVS Code統合ターミナル）を開いた
  • node -v でv18以上が表示された
  • npx -v でバージョンが表示された

STEP 03

プロジェクト作成

コマンド1本でRemotionプロジェクトが生成される。テンプレートを選んでスタート。

① プロジェクトを作成したいフォルダに移動

cd ~/Desktop # 任意の場所でOK

② Remotionプロジェクトを初期化

npx create-video@latest my-video
📋 途中で聞かれること

テンプレート選択が出る。初回は Hello World を選択してEnter。
パッケージマネージャーはnpm / pnpm / bunのどれでもOK。

③ プロジェクトフォルダに入る

cd my-video

④ 依存パッケージをインストール

npm install
  • npx create-video@latest が完了した
  • プロジェクトフォルダ（my-video）が作成された
  • npm install が完了した

STEP 04

Remotion Studio 起動

Remotion Studioはブラウザ上で動くリアルタイムプレビュー環境。コードを変更すると即座に映像が更新される。

① Studioを起動

npx remotion studio
🌐 ブラウザで開く

自動でブラウザが開く。開かない場合は http://localhost:3000 にアクセス。
左側にコンポジション一覧、右側にプレビューが表示される。

② 最初の動画を確認

src/HelloWorld/index.tsx 
import { useCurrentFrame, interpolate } from 'remotion'

export const HelloWorld = () => {
  const frame = useCurrentFrame()
  const opacity = interpolate(frame, [0, 30], [0, 1])

  return (
    <div style={{ opacity, color: 'white', fontSize: 80 }}>
      Hello World! 👋
    </div>
  )
}

③ 試しに数字を書き換えてみる
30を10に変更してファイルを保存 → プレビューが即更新されるのを確認

  • Remotion Studioがブラウザで開いた
  • HelloWorldの動画が再生できた
  • コードを変更してリアルタイム更新を体験した

STEP 05

Codexと接続

RemotionはCodexやCodexとの連携を公式に推奨している。プロンプトだけで映像を作る最速ルート。

① Remotion Skillsをインストール（推奨）

npx -y skills@latest add remotion-dev/skills -g -y
💡 Remotion Skills とは

AIエージェントにRemotionの使い方を教えるコンテキストファイル。これを入れるとCodexやCodexの精度が大幅に向上する。

② Codexからプロンプトで動画を作る

💬 CLAUDE CODE PROMPT EXAMPLE
「中央に白いテキスト『Hello AI』を表示して。0フレームから徐々にフェードインし、1秒でズームインしながら現れる。背景はダークブルーのグラデーション。30fps、5秒の動画。」

③ 動画をMP4として出力

npx remotion render HelloWorld out/video.mp4
🎬 出力完了

プロジェクトフォルダ内の out/video.mp4 に動画が生成される。
解像度・フレームレート・コーデックはすべてコードで制御できる。

  • Remotion Skillsをインストールした
  • CodexからRemotionを操作してみた
  • MP4ファイルが出力された
  • 🎉 最初のAI生成動画が完成！
